Exact Diagonalization

This technique is a brute-force approach to numerical simulations of condensed matter systems, because it constructs the Hamiltonian operator in a suitable vector space, followed by a straightforward diagonalization in order to access the spectrum.

Mathematically, exact diagonalization corresponds to finding solutions of the stationary Schrödinger equation in Dirac notation

H^∣ψn⟩=En∣ψn⟩\hat H \ket{\psi_n} = E_n\ket{\psi_n}H^∣ψn​⟩=En​∣ψn​⟩

Note that the Hamiltonian can be represented as a matrix in an arbitrary basis {ϕn}\{\phi_n\}{ϕn​}, with matrix elements Hmnϕ=⟨ϕm∣H^∣ϕn⟩H_{mn}^\phi = \braket{\phi_m | \hat H|\phi_n}Hmnϕ​=⟨ϕm​∣H^∣ϕn​⟩. The Hamiltonian in this notation is simply

H^=Hmnϕ∣ϕm⟩⟨ϕn∣.\hat H = H_{mn}^\phi\ket{\phi_m}\bra{\phi_n}.H^=Hmnϕ​∣ϕm​⟩⟨ϕn​∣.

The transverse-field Ising model (TFIM)

The TFIM is the quantum analog of the classical (Ernst) Ising model, motivated from the study of ferromagnetism in statistical mechanics.

The Hamiltonian reads

H^=J∑⟨i,j⟩S^rizS^rjz+h∑iS^rix\hat H = J\sum_{\braket{i,j}}\hat S^z_{\bm r_i}\hat S^z_{\bm r_j} + h \sum_{i}\hat S^x_{\bm r_i}H^=J⟨i,j⟩∑​S^ri​z​S^rj​z​+hi∑​S^ri​x​

in which ri\bm r_iri​ denotes some position in a Bravais lattice, and S^rik\hat S^k_{\bm r_i}S^ri​k​ is the local spin-1/2 operator along the kkk direction. J denotes the binding energy between adjacent spins (⟨i,j⟩\braket{i,j}⟨i,j⟩ denotes a constrained sum over nearest neighbors only), and hhh is the magnetic field amplitude associated with the Zeeman coupling. The spin operators satisfy the same algebraic commutation relations as orbital angular momentum, i.e.

[S^i,S^j]=iℏεijkS^l[\hat S^i,\hat S^j] = i\hbar\varepsilon_{ijk}\hat S^l[S^i,S^j]=iℏεijk​S^l

and they can be represented in terms of the Pauli matrices

S^i≃iℏ2σi,\hat S^i \simeq \frac{i\hbar}2\sigma^i,S^i≃2iℏ​σi,

The power method

Given a n×nn\times nn×n Hermitian matrix AAA, we want to find mmm extremal eigenvalues and -vectors.

Develop a power method strategy to get the job done.

  1. Choose a random starting vector b0b_0b0​
  2. Construct bk+1=Abk∣Abk∣b_{k+1} = \frac{A b_{k}}{|Ab_{k}|}bk+1​=∣Abk​∣Abk​​
  3. Repeat 2. until ϵ<∣bk+1−bk∣\epsilon < | b_{k+1}-b_{k} |ϵ<∣bk+1​−bk​∣, then compute λ=bk+1∗⋅(Abk+1)\lambda = b_{k+1}^*\cdot (A b_{k+1})λ=bk+1∗​⋅(Abk+1​)
  4. After convergence, set A→A−bbTA\rightarrow A-bb^TA→A−bbT, set m=2m=2m=2 and start from 1.

The (Cornelius) Lanczos algorithm is a substantially refined version of the power method, and presently the best method to solve the eigenvalue problem of Hermitian matrices.
